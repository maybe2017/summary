### 坑爹的问题
1. 正在运行的容器重新打包为镜像时，不会包含宿主机挂载目录（volume/bind mount）中的文件。【docker commit 只保存容器的可写层（容器内修改的文件）】
    ```
    宿主机: /home/yangdingkang/model/  ← 实际文件在这里
              ↓ (挂载)
    容器:    /workspace/               ← 只是"视图"，不占容器空间
                ↓ (docker commit)
    新镜像:                            ← 不包含 /workspace 里的文件
    ```
2. 解决方法：需要把宿主机挂载目录的文件 拷贝到容器内非挂载目录的 另一个目录，然后打包镜像才行。。。
    ```
    # 进入容器
    docker exec -it tensorrt_c_plus_1 /bin/bash

    # 复制挂载目录的内容到容器内其他位置
    cp -r /workspace/* /opt/my_models/

    # 然后提交镜像
    docker commit tensorrt_c_plus_1 my_tensorrt_image:latest
    ```
